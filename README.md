### 概述

本插件支持在您不想要打开[Discuz!](http://www.discuz.net/thread-3825961-1-1.html)的PC版的时候可以专业的显示一个二维码及一段文字描述

### 安装

> http://addon.discuz.com/?@closepc.plugin

### 特性

* 支持设置二维码链接
* 支持设置二维码标题
* 支持设置二维码描述
* 支持设置SEO标题
* 支持设置SEO描述
* 支持设置防屏蔽关键词
